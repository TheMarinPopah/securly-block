ALWAYS PUT THE URL INTO THE URL BOX IF THE SITE IS BLOCKED!!!

⚠ Do not add www to websites as this will cause Securly to block them immediately

⚠ A simple refresh can remove Securly block if it pops up.

These sites block iframes and won't load:

  • google.com
  • youtube.com
  • facebook.com
  • instagram.com
  • twitter.com / x.com
  • reddit.com
  • microsoft.com
  • tiktok.com
  • netflix.com
  • amazon.com
  • nytimes.com
  • bbc.com
  • github.com
  • discord.com
  • snapchat.com

Sites like neal.fun, Wikipedia, and most small/indie sites work fine.
